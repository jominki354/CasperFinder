# UIUX (디자인 DNA)

## Theme: 화이트 심플 모드
- **배경**: `#FFFFFF` (BG) / `#F5F5F5` (BG_CARD 대체 영역)
- **카드 배경**: `#FFFFFF` (BG_CARD)
- **텍스트**: `#333333` (TEXT) / `#666666` (TEXT_SUB) / `#999999` (TEXT_MUTED)
- **구분선**: `#E0E0E0` (DIVIDER) / `#D0D0D0` (BORDER)
- **브랜드 강조**: `#0052CC` (PRIMARY, 블루) / `#003D99` (PRIMARY_HOVER)
- **액센트**: `#5B6EF5` (ACCENT, 인디고) / `#4A5EE0` (ACCENT_HOVER)
- **성공/경고/오류**: `#16A34A` (SUCCESS) / `#FF9800` (WARNING) / `#DC2626` (ERROR)
- **호버**: `#F0F0F0` (BG_HOVER)

## 레이아웃
- **해상도**: 1280×720 고정 (리사이즈 비활성화)
- **좌측 사이드바**: 고정 네비게이션 (차량검색, 조건설정, 설정) + 알림 뱃지
- **전역 상단바 (Persistent Header)**:
    - 좌측: 경과 타이머 `00:00:00` + `[검색중]` 상태 + 시작/중지 버튼
    - 중앙: `총 N대를 찾았습니다`
    - 우측: `기획전 서버` + 상태 뱃지 (`정상`/`불안정`/`장애`/`대기 중`)
    - 하단 구분선(`1px DIVIDER`)으로 콘텐츠 영역과 분리
- **정렬 영역**: 좌측 정렬 버튼 3종 (높은가격순, 낮은가격순, 생산일순)
- **필터 영역**: 우측 콤보박스 4종 (트림, 옵션, 외장색상, 내장색상)
- **카드 리스트**: 스무스 스크롤 가능한 차량 카드 목록 (10개/페이지)

## 서버 상태 툴팁
```
┌──────────────────────┐
│ [기획전 응답 상세]     │
│ (3초 전 확인됨)        │
│                      │
│ 특별기획전: 64ms      │
│ 전시차:    97ms       │
│ 리퍼브:    84ms       │
└──────────────────────┘
```
- **구현**: Pre-built `CTkFrame` (호버 시 텍스트만 갱신, 위젯 생성/삭제 없음)
- **위치**: 상단 바 아래 우측 고정 (x=1270, y=48)
- **표시/숨김**: `place()` / `place_forget()` + `lift()` Z-순서 보장

## 차량 카드 구조
```
┌──────────────────────────────────────────────┐
│ 모델명 트림명  (차량ID)            [라벨]    │
│                                              │
│ [색상칩] 외장색  출고센터  생산일  할인  최종가격│
│ [색상칩] 내장색                              │
│                                              │
│ 옵션: 스마트센스, 선루프, ...      [계약하기] │
└──────────────────────────────────────────────┘
```

## 인터랙션
- **알림 토스트**: 우측 하단 인앱 알림 (330×130px), 0.5초 페이드인, 2.5초 표시
- **알림 클릭**: 차량검색 탭 전환 + 해당 카드 1.5초 노란색 하이라이트
- **옵션 복수 선택**: 토글 방식, 선택된 항목에 `✓ ` 체크 표시
- **필터 요약 표시**: `옵션명 외 N개` 형태로 현재 상태 요약
- **카드 하이라이트**: `#FFF9C4` 배경 + ACCENT 테두리 (1.5초)
- **서버 상태 호버**: 즉시 상세 핑 정보 말풍선 표시, 마우스 이탈 시 즉시 숨김
- **검색 빈 화면**: 상태에 따라 안내 메시지 동적 전환 (검색 중 vs 대기 중)
