# CasperFinder
캐스퍼 일렉트릭 기획전 재고 모니터링 데스크톱 앱 (Python + CustomTkinter)

## 개요
현대자동차 캐스퍼 일렉트릭의 기획전(E/D/R) 페이지를 주기적으로 폴링하여,
신규 차량이 등록되면 **Windows 토스트 알림**을 보내고 **차량 카드 리스트**로 표시하는 데스크톱 앱입니다.

## 주요 기능
- **실시간 모니터링**: 5초 간격 API 폴링으로 신규 차량 즉시 감지
- **토스트 알림**: 모델/트림/가격/옵션 요약 + 클릭 시 해당 카드 하이라이트 포커싱
- **차량 카드 뷰**: 색상 칩 이미지, 출고센터, 생산일, 할인, 최종가격, 옵션 표시
- **정렬/필터링**: 가격순/생산일순 정렬 + 트림/옵션/외장/내장 색상 필터
- **옵션 복수 선택**: 드롭다운에서 토글 방식으로 다중 옵션 필터링 (체크 표시 `v`)
- **시스템 트레이**: 최소화 시 트레이 상주, 더블클릭으로 복원
- **총 발견 대수 표시**: 상단 바에 실시간 매물 건수 요약

## 기술 스택
- **Python 3.10+**
- **CustomTkinter**: GUI 프레임워크
- **pystray + Pillow**: 시스템 트레이
- **asyncio + aiohttp**: 비동기 API 폴링
- **winotify**: Windows 토스트 알림

